# Codex Theme Pack for Visual Studio Code

An unofficial port of all 43 light and dark appearance themes bundled with the Codex desktop app.

The extension preserves each source theme's editor colors and TextMate syntax rules, then fills VS Code-specific workbench, terminal, diff, diagnostic, and semantic-token gaps with colors derived from that theme.

## Install a local build

1. Run `npm run package:vscode` from the repository root.
2. In VS Code, run **Extensions: Install from VSIX...**.
3. Select `dist/codex-theme-pack-vscode-0.2.0.vsix`.
4. Run **Preferences: Color Theme** and choose any theme beginning with **Codex —**.

See the repository's `THEMES.md` for the complete list.
