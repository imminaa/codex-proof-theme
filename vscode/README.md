# Proof Light for VS Code

An unofficial port of the Codex desktop app's **Proof Light** theme.

The theme preserves the original warm canvas, soft chrome, forest-green accent, periwinkle keywords, ochre constants, and restrained neutral punctuation. Additional selection, diagnostic, diff, terminal, and semantic-token colors are derived to make the palette complete in VS Code.

## Install a local build

1. Run `npm run package:vscode` from the repository root.
2. In VS Code, run **Extensions: Install from VSIX...**.
3. Select `dist/proof-light-vscode-0.1.0.vsix`.
4. Run **Preferences: Color Theme** and choose **Proof Light**.

This is an unofficial personal-use port and is not affiliated with or endorsed by OpenAI.
